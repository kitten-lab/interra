# syntax shapes
tags begin as a lightweight custom DSL.  
a raw string-example: `NEWS*MEDIA>SKYLINE-NEWS;SKYLINE-NEWS*section>updates;`

```
### parser
``` DSL format
`A*B>C;`

- `A` = source entity  
- `*` = relationship separator  
- `B` = category / context  
- `>` = directional link  
- `C` = target entity  
- `;` = statement terminator
```
